# 🎬 Youtubebrief

**Browser for account and billing. Terminal for clean YouTube briefs.**

Youtubebrief builds CLI/MCP tools that turn explicit YouTube URLs into structured, timestamp-backed Markdown for research, notes, DevRel handoffs, and local AI-agent workflows.

## 🚀 Start here

- 📦 Public beta repo: [`youtubebrief-cli`](https://github.com/youtubebrief/youtubebrief-cli)
- 🌐 Product site: <https://youtubebrief.com>
- 🧭 CLI setup: <https://youtubebrief.com/cli>
- 🔑 Beta access: <https://youtubebrief.com/beta>

## 🧩 What we publish

- **CLI** for clean terminal workflows around YouTube briefs.
- **Local stdio MCP** for Codex, Claude Code, and other agent clients.
- **Markdown/JSONL bundle workflows** for research, notes, and RAG preparation.
